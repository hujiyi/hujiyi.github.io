聚合查询

aggregate() 是 QuerySet 的一个终止子句，意思是说，它返回一个包含一些键值对的字典。键的名称是聚合值的标识符，值是计算出来的聚合值。键的名称是按照字段和聚合函数的名称自动生成出来的。

如果你想要为聚合值指定一个名称，可以向聚合子句提供它。

分类汇总小结

  • 第一部分 Order.objects.values("stock__supplier__name"), 在annotate 之前使用的values()用于指定分类的字段

  • 第二部分 annotate( month=ExtractMonth('time_on_order'),total=Sum("total_price")) , annotate() 用于指定汇总的方法和相应的字段

  • 第三部分 .values("stock__supplier__name", "total", "month"), 在annotate 之后出现的 values 统计结果中输出的字段名。

小技巧: 在分类汇总查询后, 使用 return HttpResponse(data_supplier) 直接在浏览器输出查询结果的 JSON 数据以进行检查, 确认输出数据无误后, 再进行图表的制作
