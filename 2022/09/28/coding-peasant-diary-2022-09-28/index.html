coding peasant diary 2022 09 28

发表于2022-09-28|更新于2022-10-11|diary
|总字数:926|阅读时长:2分钟|浏览量:

昨天买的四脚贴片微动开关到货, 把25日就拆开的微软3600鼠标修好了, 在Dell 电脑上试了没有问题, 就把它装到包里, 罗技M720虽然好用, 可是那个拟肤外壳还要看看怎么处理一下, 不然总在在外面包一层纸巾, 拿着特别累。

一大早打开电脑, 打开鼠标, 没反应。昨天下班的时候电脑提示要重启更新, 觉得这个更新不知道要多长时间, 所以直接关上盖子就回去了, 难道是因为更新了没有关机? 还是因为鼠标连接过其他的电脑? 于是先把电脑里原来的已连接的蓝牙鼠标删除, 再重新添加, 居然提示无法连接, 请尝试重新连接设备,这是什么鬼啊, 难道鼠标换个微动开关, 还能导致无法连接, 那昨晚在另一台电脑也用得好好的啊。

反得重试了多次还是一样, 到网上搜索, 有人说是因为 Windows 11 升级更新的缘故, 回退到上一个版本就好了, 这怎么可能呢, 第一时间我已经把回退的东西清理干净了; 还有人说, 把网卡、蓝牙什么的驱动删除再重新安装就可以了, 不过几次尝试, 还是解决不了问题。

也许只能重新安装系统啊(是的, 我一直想重新一个干净的系统), 因为所说只有企业版的 Windows 11 才可以在安装的时候不用登录微软账号, 那就先弄一个数字激活的企业版吧。找了企业版的 Key, 断网、输入Key, 提示没有联网不能激活, 于是重新连接网络, 用 ActTool_2.4 做了 一键数字化, 完美, 看看得下找个什么时间安装系统吧。

在控制面板不小心点到了蓝牙设备, 里面显示 Bluetooth Mobile Mouse 3600 已配对, 难道鼠标可以用了? 重新打开鼠标电源, 并没有, 要不重新连接一次? 于是把这个已配对的设备删除, 再次连接, 居然成功了, 不知道这是什么奇葩操作。刚才已经做好了用一整天触控板的准备, 不过还是鼠标用起来更顺手。

接近中午的时候, 钟毓求助帮忙处理她要比赛的练习中遇到的问题, 这个东西我其实也不会, 不过后来居然把结果弄出来了, 但是不小心也留下了一个坑, 等后来做到显示图表的时候才发现, 计算结果要保存为数字而不是字符类型。

搞完其他事情, 继续昨天的搭框架工作, 然后体会到自己搭建一个 Spring Boot 的框架并不是一个容易的事, 而且是在自己并不掌握 Spring Boot 的情况下, 纯粹就是好高骛远的行为, 路要慢慢的走, 所以还是先从基础开始吧。

ThinkBook 16+自带的联想电脑管家 居然有一个 开机画面定制 功能, 也许联想也知道自己电脑的那个开机画面丑到爆了, 试了一下, 做了一个开机画面, 不能说很好吧, 至少没有自带的那个那么突兀了。

下午的时候试一下用 Hyper-v 来安装 Linux Mint 21, 简直是太卡了, 感觉已经没有时间等到安装结束了, 还是先把它停了吧。哪天得空了再说, 或者还是用 VMware 比较好。
