使用 PyCharm Community 开发 Django 应用程序 第一部分 - 项目和创建与配置

发表于2022-08-29|更新于2024-10-15|django
|总字数:2.1k|阅读时长:10分钟|浏览量:

开发环境：

  • Microsoft Windows [版本 10.0.14393], locale zh-CN

  • Python 3.6.4

  • PyCharm Community Edition 2021.3.3

  • Visual Studio Code 1.70.0 (system setup), 64-bit edition

PyCharm Community 新建项目中没有 Django的项目类型, 只能使用命令方式创建项目, 所以需要先在电脑上安装 Django。

安装 Django

打开 命令提示符 窗口, 进入 Documents(文档) 文件夹

之后创建的 Django 项目将保存在 Documents(文档) 文件夹中

使用 pip 安装 Django : pip install django, 安装过程如下:

查看安装好的 Django 的版本

不同版本的 Python 安装对应版本的 Django, 当前电脑安装的是 Python 3.6.4, 对应的是 Django 3.2

Django 与 Python 的版本对应关系

Django 版本 Python 版本
2.2 3.5，3.6，3.7，3.8（2.2.8 添加），3.9（2.2.17 添加）
3.0 3.6，3.7，3.8，3.9 （3.0.11 添加）
3.1 3.6，3.7，3.8，3.9（3.1.3 添加）
3.2 3.6, 3.7, 3.8, 3.9, 3.10 (在 3.2.9 中就已经加入了)
4.0 3.8，3.9，3.10

创建 Django 项目

使用命令创建项目

在命令提示符中输入命令: django-admin startproject <项目名称>

此处创建的项目名称为: mystore

离线安装项目需要的包

打开一个新的 命令提示符窗口, 并进入离线安装包所在的位置。

  1. 安装 MySQL 数据库驱动
  1. 安装 PyeCharts 图表类库

此处创建项目没有使用虚拟环境, 如果使用了虚拟环境, 需要激活虚拟环境后再执行安装离线包的命令, 为减少操作环节, 可以将离线包复制到项目文件夹, 然后进行安装, 减少步骤, 提高安装效率。
