使用 insertAdjacentHTML 添加 HTML 代码

insertAdjacentHTML() 方法将指定的文本解析为 Element 元素，并将结果节点插入到 DOM 树中的指定位置

  • 'beforebegin'：元素自身的前面。

  • 'afterbegin'：插入元素内部的第一个子节点之前。

  • 'beforeend'：插入元素内部的最后一个子节点之后。

  • 'afterend'：元素自身的后面。
