coding peasant diary 2022 10 11

发表于2022-10-11|更新于2022-10-11|diary
|总字数:1.5k|阅读时长:4分钟|浏览量:

昨天早餐的糯米饭团味道很不错, 所以今天继续买。 虽然每天早上有一个阿姨也在路口卖更便宜的糯米饭, 可是她那个有很明显的不知道是香油还是什么的其他味道, 反而不怎么感兴趣。

昨晚把下了决心, 重新安装了电脑的系统, 那个ISO镜像文件虽然已经提前做了更新, 可是安装完之后发现系统的版本还是 22621.607, 于是装好了, 又提示要升级到 22621.608, 好吧, 下次还是要重新下载新版本的才行。

安装过程基本顺利, 跳过安装的联网步骤, 目前只找到修改注册表的方式, 另外几种方法不知道是不是因为新版本的问题用不了。系统更新的时候, 明明已经完成了更新, 但还是离有好几个项目在更新列表中, 并且出现了更新错误:0x80248007, 最后在微软的官网找到了解决的办法, 但目测处理之后的更新过程, 大概只是把那几个实际早就完成了更新的项目从列表中清除而已。

接上第二个显示器之后, 在右下角的 快速设置 区域中, 居然找不到设置扩展显示的那个项目, 后来弄了好久, 总算把它搞出来了, 感觉Windows 11 在控制这一块的改动很大, 可是很多常用的设置却很难找得到。其实重新安装系统并没有花太多时间, 但处理安装后的一些小问题却搞了很久。

昨晚遇到的另外一个问题是 hujiyi.github.io 的仓库, 这次安装系统之后使用的用户名和上次开机设置的并不一样, 所以使用 hexo 处理并上传静态页文件的时候出现了错误, 总是提示没有设置 git username, 明明已经做了本地的设置, 后来又发现因为项目文件夹的所有者发生了变化, 导致无法 git commit, 感觉真不容易啊。

于是只好新建一个本地仓库文件夹, 再把旧的文件复制过来, 结果发现还是不行。 最后只好把之前压缩备份的本地仓库项目解压, 再把相应的文件全部复制进去替换, 总算是解决了这个问题。

弄完了发现已经23点多了, 时间过得真快啊。 关了电脑, 随后用手机上去一看, 默认的文章图片却是 404, 看来还是有一个地方没弄对。

今天到了公司就先弄这个部分吧, 从前一次正确的备份中把相应的配置文件替换进去, 本地测试了一下没有问题, 于是上传到 github。

上传成功后又一时手痒, 把 pages 的仓库设成了私有仓库, 结果发现, 私有仓库并不免费提供 pages 功能, 所以要么付费, 要么改回公开的仓库。

因为遇到了本地仓库可能会出意外的状况, 所以还是把源文件也上传到 github 中比较保险, 开始是先找了一个以前旧的仓库, 打算直接覆盖的, 结果并不成功。那还是用简单粗暴一点的吧, 直接把那个仓库删了, 再建另外一个好了。

以后要记得: 每次写好博客, 除了上传静态页, 还要上传源文件仓库。

github 的东西虽然不多, 但是也花了一些时间, 接下来就是安装 Java 的开发环境, 从官网下载了最新版的 IDEA 2022.2.3, 破解文件之前已经下载好了, 虽然破解文件是之前下载的, 支持到 IDEA 2022.2.2, 但是看了一下那个破解的过程, 感觉新版的应该也没有什么问题, 所以还是把它装上了。不过随后因为其他情况打开环境变量设置界面的时候, 发现多了一个窗口都放不完的破解相关的变量, 这也太可怕了。

以前黄伟因为某个项目给大家申请过企业邮箱, 这个邮箱可以做为教育邮箱使用, 当时还用它用申请了免费的教育账号, 不过后来被收回、注销了。在网上找了一圈, 发现已经很难再有能免费申请到的教育邮箱, 某宝上卖的很不便宜, 而且还只能用一年。

研究了很久, 决定用自己的校牌去申请一个教育账号, 不知道能不能得到认证。

在公司摸鱼了几天之后, 今天又收到新任务了, 十一点多到会议室, 研究卫健委提交的人事信息系统优化清单, 听了需求, 感觉现在开发系统, 其实只有两个功能是最核心的: 批量上传Excel数据和批量导出Excel数据, 所以花了几十万做的系统, 就是为了方便大家导一个 Excel 而已。

需要的优化的项目不少, 当前先要完成的是红色和绿色, 负责讲解的美女说等她先出一个原型。

上次健哥给我的那个项目代码好象是有些问题的, 还是要等他分给我新的仓库。前段时间, 还真是把这个人事系统都用了一遍, 也不知道是不是为了这次做的准备(其实是为了软著)

还是先把开发环境弄好吧, 安装配置隧道, 修改配置文件, 添加 git 要忽略的文件, 给前端登录界面加上密码, 也顺便做个忽略。好吧,准备就绪, 下一步从哪里开始呢?

今天的天气真是好, 天空很蓝, 一丝云也没有; 温度很合适, 不冷也不热; 微风轻轻地吹过, 一切都是很舒适的样子。
